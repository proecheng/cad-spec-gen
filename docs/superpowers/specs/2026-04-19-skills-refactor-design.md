def _render_skill_row(skill: dict) -> str:
    """渲染单行 skill 表格。description 取 skill.json 的 description，裁剪到 100 字以内。"""
    desc = skill.get('description', '')[:100]
    if len(skill.get('description', '')) > 100:
        desc += '...'
    return f"| {skill['name']} | `{skill['trigger']}` | {desc} |"
```

### 3.4 确定性

- **不含** timestamp / git rev / 任何 volatile 数据
- 纯函数：相同 `skill.json` → 相同 `AGENTS.md` 字节
- 连跑 2 次 `python scripts/dev_sync.py` 第二次不产生 diff

### 3.5 集成到现有 pre-commit

`.pre-commit-config.yaml` 现有 hook `dev-sync` 已 `always_run: true` + `pass_filenames: false`——**零改动**即可自动守护 `AGENTS.md`：
- 改 `skill.json` → dev_sync 跑 → 生成新 `AGENTS.md` → git diff 非空 → pre-commit exit 1 → 强制 `git add AGENTS.md` 重 commit
