### Step 2.2 — 写一次性脚本 `tmp/p2_apply_noqa.py`（A-2，5 min；L3-2 闭合，推荐方式 1）

- [ ] **S-2.2.1** Create `tmp/p2_apply_noqa.py`（不入仓）：
  ```python
  """一次性 P2 Step 2.2 工具：给 25 个 codegen 脚手架加 file-scoped noqa F403/F405。

  位置：docstring `"""` 结束之后、`import cadquery as cq` 之前。
  幂等：若文件已含 `# ruff: noqa: F403, F405` 则跳过。
  """
  from pathlib import Path
  import re

  TARGETS = list(Path("cad/end_effector").glob("*.py")) + list(Path("cad/lifting_platform").glob("*.py"))
  NOQA = "# ruff: noqa: F403, F405\n"
  count_added = 0
  count_skipped = 0
  for f in TARGETS:
      src = f.read_text(encoding="utf-8")
      if NOQA in src:
          count_skipped += 1
          continue
      # 找 docstring 闭合 \"\"\" 后第一个 import
      m = re.search(r'(\"\"\"\n[\s\S]*?\"\"\")\n+(import |from )', src)
      if not m:
          print(f"SKIP (no docstring+import pattern): {f}")
          continue
      patched = src[:m.end(1)] + "\n\n" + NOQA + src[m.end(1)+1:]  # 在 docstring 后插
      # 重写：在 docstring 后 import 前插 noqa
      fixed = re.sub(
          r'(\"\"\"\n[\s\S]*?\"\"\")(\n+)(import |from )',
          r'\1\n\n' + NOQA + r'\3',
          src,
          count=1,
      )
      f.write_text(fixed, encoding="utf-8")
      count_added += 1
      print(f"ADDED noqa to: {f}")
  print(f"\nSummary: added={count_added} skipped={count_skipped} total_targets={len(TARGETS)}")
  ```
