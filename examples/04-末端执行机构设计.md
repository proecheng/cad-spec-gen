### 4.1 四工位旋转法兰总成

#### 4.1.1 结构参数

| 参数 | 设计值 | 说明 |
|------|-------|------|
| 旋转分度 | 90° × 4工位 | |
| 驱动方式 | Maxon ECX SPEED 22L + GP22C行星减速器（53:1） | 输出轴Φ8mm，法兰连接 |
| 定位精度 | 重复定位优于0.2° | 弹簧销 + 伺服闭环双重保证 |
| 切换时间 | ＜1.5s（90°切换） | S形速度曲线 |
| **法兰本体直径** | **Φ90±0.1mm** | 圆盘部分外径 |
| **悬臂长度** | **40mm**（从法兰边缘到工位安装面中心） | 4根等长，90°等分 |
| 法兰厚度 | 30±0.5mm（含PEEK段5mm） | 铝合金段25mm + PEEK段5mm |
| 法兰材质 | 7075-T6铝合金 + PEEK绝缘隔离段 | 轻量化 + 电气绝缘 |
| 电机+减速器重量 | 约180g | ECX 22L(70g)+GP22C(110g) |
| 电机额定扭矩 | 0.027Nm（减速后1.43Nm） | 堵转2.85Nm |
| 电机外径 | Φ22mm × 长73mm（含减速器） | 安装于法兰背面中心 |
| **轴-法兰连接方式** | **Φ8mm输出轴 → Φ8H7/k6过盈配合 + M4轴端压紧螺栓** | 传递堵转2.85Nm |
